# Task Manager

Тестовое задание на позицию Frontend Developer.

## Описание проекта

Task Manager — fullstack-приложение для управления задачами.

Основной упор сделан на frontend-часть приложения с использованием React и TypeScript. Пользователь может:

- просматривать ближайшие задачи;
- просматривать задачи на выбранный день;
- просматривать задачи за произвольный период;
- фильтровать задачи по статусу и приоритету;
- сортировать задачи по времени и приоритету;
- создавать задачи;
- редактировать задачи;
- удалять задачи.

Приложение построено по архитектуре Feature-Sliced Design (FSD).

## Основные страницы

### Главная страница

- список ближайших задач;
- краткая информация о задачах;
- навигация по приложению.

### Расписание на день

- задачи на выбранный день;
- фильтрация и сортировка;
- топ задач на неделю;
- топ задач на месяц.

### Задачи за период

- выбор диапазона дат;
- просмотр задач за период;
- фильтрация и сортировка.

### Создание и редактирование задачи

- форма на React Hook Form;
- валидация через Zod.

## Запуск проекта

Для запуска требуется установленный Docker и Docker Compose.

В корне проекта выполните:

```bash
docker compose up -d --build
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

API:

```text
http://localhost:4000/api/v1
```

## Тестирование

В проекте присутствуют демонстрационные тесты для:
